Drop all photos and the video directly into this folder
(the same folder as index.html) using these EXACT filenames —
they match config.js. If you rename a file, update the matching
field in config.js too.

Images:
  hero.jpg              -> Home hero, full-width
  venue.jpg              -> Venue photo, full-width
  bride.jpg              -> Couple tab, bride portrait
  groom.jpg              -> Couple tab, groom portrait
  story-1.jpg            -> "How We Met" story photo
  story-2.jpg            -> "The Proposal" story photo
  event-mehendi.jpg      -> Mehendi event card
  event-sangeet.jpg      -> Sangeet event card
  event-wedding.jpg      -> Wedding ceremony event card
  event-reception.jpg    -> Reception event card

Video:
  love-story.mp4  -> the video shown on Home (click-to-play, not autoplay)
  poster.jpg      -> still frame shown before the video is played

Recommended: photos at least 1600px wide for crisp full-width display;
compress the video (e.g. with HandBrake) to stay well under 50MB.

Until a file is added, that spot shows a labeled placeholder box —
nothing looks broken either way.
